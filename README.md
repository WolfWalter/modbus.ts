# Modbus.ts

[Modbus](http://www.modbus.org/) application protocol written in [TypeScript](https://www.typescriptlang.org/).

## Developer

```Shell
# Install dependencies.
$ yarn install

# Clean distribution directory.
$ yarn run clean

# Clean Node modules directory.
$ yarn run distclean

# Run tests.
$ yarn run test

# Run tests with coverage.
$ yarn run coverage

# Run linter.
$ yarn run lint

# Package library.
$ yarn run pack
```
